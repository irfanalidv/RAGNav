# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | -------------------- |
| 0.4.x   | :white_check_mark: |
| < 0.4   | :x:                |

## Reporting a Vulnerability

Please report security issues privately using GitHub's "Report a vulnerability"
(Security tab) or by emailing irfanali29@hotmail.com. I aim to acknowledge within
72 hours and to share a remediation timeline after triage. Please do not open
public issues for security-sensitive reports.
